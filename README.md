# 婴儿喂养记录管理 Skill

用于 Trae IDE 的婴儿喂养记录管理技能，通过自然语言对话即可记录和查询婴儿的饮食、便便、护理信息。

## 功能

- **记录喂养**：母乳、配方奶、米粉摄入量，补剂（AD/D3）、体温
- **记录便便**：量、颜色、形状
- **记录护理**：洗澡、排气操、被动操、外出等
- **查询统计**：按日期查询、每日总结、最近记录、关键词搜索

## 使用方式

在 Trae 对话中直接描述你的需求，例如：

```
记录一下：9点母乳110ml，配方奶0，米粉2.5g，吃了AD
```

```
今天上午拉了便便，量中等，颜色黄，形状正常
```

```
今天下午做了排气操
```

```
今天喂养情况怎么样？
```

```
最近3天吃了多少奶？
```

```
搜索一下所有洗澡的记录
```

## 数据存储

所有记录保存在 `baby_feeding_data.xlsx` 中（首次使用时自动创建），包含三个工作表：

| 工作表 | 内容 |
|--------|------|
| 饮食 | 喂养记录（时间、奶量、米粉、补剂、体温、备注） |
| 便便 | 便便记录（时间、量、颜色、形状、备注） |
| 护理 | 护理记录（日期、时段、项目、时间段、备注） |
